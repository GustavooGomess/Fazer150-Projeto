# Fazer150-Projeto
Landing page moderna para a Yamaha Fazer 150, focada em design esportivo e alta performance. O projeto utiliza HTML5 e CSS3 (Flexbox/Grid), apresentando tipografia de impacto, botões interativos com transições suaves e composição de camadas com gradientes sobre imagens HD, garantindo uma interface imersiva, tecnológica e de alta conversão.
